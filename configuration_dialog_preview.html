✅ Configuration Dialog Preview - User-Configurable Settings Only

This preview demonstrates the configuration dialog with all user-configurable settings:

  • ✅ Core authentication settings (Token, Endpoint, Authentication Method)
  • ✅ Product activation toggles (OSS, Code, IaC, Security, Quality)
  • ✅ CLI and path configuration
  • ✅ Security settings (Insecure mode, Trusted folders)
  • ✅ Filter severity and display options
  • ✅ Feature toggles (Learn, Quick Fix, Browser Actions, Delta Findings)
  • ✅ Advanced settings (Snyk Code API, CLI parameters, Environment variables)
  • ✅ Folder-specific settings with proper UI for scan config (no JSON editing!)

Excluded fields: Global Organization (use folder-level), Automatic Authentication (managed by IDE), SAST Settings (auto-populated), and all system fields (Integration info, Device ID, OS details, Protocol version).

Scan Command Config: Pre/Post scan commands can be configured per product with proper UI controls instead of JSON editing.
